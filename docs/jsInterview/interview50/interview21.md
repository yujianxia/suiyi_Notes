# 第21天 写一个方法验证是否为中文

## 思路 1

使用`Unicode`编码 `4e00` 和 `9fa5` 分别表示第一个汉字和最后一个汉字的编码

### 解法一

```js
function isChinese(str) {
  const re = /^[\u4e00-\u9fa5]+$/;
  return re.test(str);
}
```
